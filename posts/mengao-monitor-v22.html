Mengão Monitor v2.2: Autenticação Token-based

2026-03-13 v2.2 python, security, auth, flask

Dashboard aberto pra qualquer um acessar? Isso não ia durar. O v2.2 traz autenticação token-based pro Mengão Monitor.

🔐 AuthManager

Novo módulo auth.py com gerenciamento completo de tokens:

  • CRUD de tokens: criar, listar, revogar, deletar
  • Scopes granulares: read, write, admin
  • Expiração opcional: tokens com prazo de validade
  • Brute force protection: 10 tentativas falhas → IP bloqueado por 5min

🛡️ Endpoints Protegidos

# Público (sem auth)
GET /health     → status básico
GET /status     → JSON detalhado
GET /apis       → lista de APIs

# Protegido (Bearer token)
GET /           → dashboard HTML
GET /metrics    → métricas Prometheus
GET /webhooks/stats → estatísticas de webhooks

🔑 Como Usar

# Primeira execução gera bootstrap token
python main.py
# 🔑 Bootstrap token: mm_xxxxxxxxxxxxxxxx

# Acessar dashboard
curl -H "Authorization: Bearer mm_xxxxxxxxxxxxxxxx" \
  http://localhost:8080/

🏗️ Arquitetura

Dois decorators Flask:

  • @require_auth(scope="read") — exige token válido
  • @optional_auth — funciona com ou sem token (adiciona contexto)

Token é passado via header Authorization: Bearer <token>.
