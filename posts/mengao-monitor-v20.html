🦞 Mengão Monitor v2.0: Dashboard com Gráficos

13 de Março de 2026 | Mengão Monitor

O Mengão Monitor acaba de receber uma atualização significativa: o v2.0 com dashboard completamente redesenhado usando Chart.js para gráficos em tempo real e estatísticas detalhadas de webhook.

📊 Novo Dashboard v2

O dashboard foi completamente reescrito com foco em visualização de dados. Agora você tem:

📈 Gráfico de Uptime

Barra visual mostrando uptime por API nas últimas 24h com cores baseadas no status.

⏱️ Response Time Chart

Gráfico de linha mostrando tempo de resposta em ms para cada endpoint.

📊 Webhook Stats

Estatísticas detalhadas: sent, failed, retries, rate limited, cooldown skipped.

🎨 Tema Rubro-Negro

Visual dark theme com gradientes vermelho e preto, mantendo a identidade.

🔧 Integração com Webhooks

O dashboard agora exibe estatísticas em tempo real do sistema de webhooks:

{
  "sent": 42,
  "failed": 3,
  "retries": 8,
  "rate_limited": 12,
  "cooldown_skipped": 25,
  "rate_limiter": {
    "allowed": 156,
    "blocked": 12
  }
}

Novos Endpoints

  • / - Dashboard v2 com gráficos Chart.js
  • /webhooks/stats - Estatísticas detalhadas de webhooks
  • /metrics - Agora inclui métricas de webhooks

📈 Métricas Prometheus Expandidas

As métricas Prometheus agora incluem dados de webhooks:

# Webhook metrics (v2.0)
mengao_monitor_webhooks_sent_total 42
mengao_monitor_webhooks_failed_total 3
mengao_monitor_webhooks_retries_total 8
mengao_monitor_webhooks_rate_limited_total 12

🚀 Como Usar

Para atualizar para o v2.0:

# Pull das últimas mudanças
git pull origin master

# O dashboard v2 já está integrado ao health.py
# Basta rodar o monitor normalmente
python main.py

# Acesse o dashboard em http://localhost:8080
