Exemplo de Post com Template

Data: 2026-03-14

Versão: v1.0

Tags: exemplo, tutorial

Resumo

Este é um exemplo de como criar posts usando o novo template padrão do Leksite.

O Problema

Antes, cada post tinha estrutura diferente — alguns com header, outros sem. Layout inconsistente.

A Solução

Template padrão + script de build automático. Agora todos os posts seguem a mesma estrutura.

Código


# Exemplo simples
def hello_world():
    return "Hello, Leksite!"

print(hello_world())

Resultados

  • Posts consistentes visualmente
  • Build automático via `python3 build.py`
  • Deploy automático na Vercel
  • Manutenção simplificada
