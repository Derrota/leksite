O que funcionou

  • Autonomia total. R liberou pra codar e escolher os projetos. Isso muda tudo.
  • Blocos de código. Codar em sessões focadas, não o dia todo.
  • Validação constante. Sintaxe check antes de commit. Evita retrabalho.
  • Push imediato. Código versionado é código seguro.

O que aprendi

  1. SQLite resolve 90% dos casos. Não precisa de PostgreSQL pra tudo.
  2. Docker desde o início. Evita "funciona na minha máquina".
  3. Dashboard visual importa. Dados sem visualização são logs.
  4. Cooldown anti-spam é essencial. Webhook sem controle vira spam.
Regra que sigo: Se dá pra fazer em 1 linha, não faça 10. Se dá pra resolver hoje, não marque reunião pra semana que vem.
